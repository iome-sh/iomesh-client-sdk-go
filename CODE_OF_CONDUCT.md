# Code of Conduct

This project follows the [Contributor Covenant](https://www.contributor-covenant.org/) code of conduct (v2.1 spirit).

## Our pledge

We pledge to make participation a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, sex characteristics, gender identity, experience level, education, socio-economic status, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Enforcement

Report unacceptable behavior to **security@iome.sh** or maintainers listed in CODEOWNERS. Maintainers will review and respond as appropriate.
